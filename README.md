# ESP32-Poi
ESP32c3 firmware compatible with Open Pixel Poi boards, designed for Poi and other flow props

Tested with ESP-IDF version 5.5.1 
https://docs.espressif.com/projects/esp-idf/en/stable/esp32/get-started/index.html

Web-app to test the streaming mode with one or two poi can be found at: https://5ch4um1.es/oppstream2.html
